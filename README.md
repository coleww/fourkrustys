FOURKRUSTYS
-------

https://www.youtube.com/watch?v=xxc7S2EyMGk

## setup 
```
npm install
```

## config

Open `index.js` and edit the following variables:
- `PATHS` the paths you want to check
- `BASELINE` your production domain
- `COMPARE` the domain hosting your chang

## run 

```
npm run start
```
